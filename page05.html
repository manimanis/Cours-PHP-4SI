Se connecter au serveur MySQL

mysql_connect($host, $user, $password)

Paramètres

$host (chaine) : Adresse du serveur MySQL

$user (chaine) : Nom de l'utilisateur

$password (chaine) : Mot de passe

Résultat

Descripteur de connexion

Exemple

Sélectionner une base de données de Travail

mysql_select_db($base_de_donnees)

Paramètres

$base_de_donnees (chaine) : Nom de la base de données de travail

Résultat

TRUE en cas de succès, FALSE en cas d'échec

Exemple

Renvoyer une requête de sélection

$f = mysql_query($req)

Paramètres

$req (chaine) : Requête de sélection (SELECT ... FROM ...)

Résultat

$f (resource) : Une ressource représenant l'ensemble d'enregistrements renvoyés par la requête de sélection

Exemple

Tester le nombre de résultats retournés par une requête de sélection

$nr = mysql_num_rows($f)

Paramètres

$f (resource) : Une ressource représenant l'ensemble d'enregistrements renvoyés par la requête de sélection

Résultat

$nr (entier) : Le nombre d'enregistrements renvoyés par la requête de sélection

Exemple

$ligne = mysql_fetch_assoc($f)

$ligne (tableau) : Retourne un tableau associatif (l'index des éléments est le nom des champs) ou
FALSE si fin de fichier

Résultat

$f (resource) : Une ressource représenant l'ensemble d'enregistrements renvoyés par la requête de sélection
