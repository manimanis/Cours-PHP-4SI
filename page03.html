Ouvrir le site Web

Ouvrir l'éditeur de pages Web FrontPage.

Cliquer sur l'icone Ouvrir > Ouvrir le site... dans la barre d'outils.

Sélectionner le dossier de travail

Sélectionner le dossier de travail dans la racine du serveur Web (C:\EasyPHP\www\Seance1 par exemple).

Accepter que FrontPage ajoute des informations à votre dossier : Cliquer sur Oui.

FrontPage affiche alors la liste des dossiers. Tous vos fichiers et images devront être enregistrés dans ce dossier.

Créer une nouvelle page Web

Cliquer avec le bouton droit de la souris sur la racine du site. Puis, Nouveau > page vierge

Donner un nom au fichier (exercice02.html par exemple).

Créer un formulaire

Utiliser la barre d'outils formulaires pour créer un formulaire.

La barre d'outils formulaires se trouve dans le menu Insertion > Formulaire > Faire glisser pour détacher ce menu

Ne pas oublier de donner des noms aux contrôles du formulaire (j, m, a dans l'exemple).

Indiquer la destination du formulaire

Cliquer avec le bouton droit au dessus du formulaire Propriétés du formulaire...

Sélectionner l'option Envoyer à (autre) Script personnalisé ISAPI, NSAPI, CGI ou ASP.

Donner un nom au formulaire (f par exemple).

Cliquer sur le bouton Options... pour indiquer la destination du formulaire.

Traiter les données du formulaire

Créer une page vierge. Nommer avec l'extension php (exercice02.php par exemple).

Confirmer l'attribution d'un nouveau nom : Cliquer sur Oui.

Cliquer avec le bouton droit au dessus du nouveau fichier php > Ouvrir avec... > Notepad++.

Taper le script PHP.
