Ouvrir la fenêtre d'administration

Cliquer sur avec le bouton droit de la souris sur l'icone EasyPHP > Administration.

Ouvrir l'application phpMyAdmin

Cliquer sur le lien Administer MySQL....

Il est également possible d'accéder à la page d'administration en tapant l'adresse : http://127.0.0.1/home/.

Créer la base de données

Taper le nom de la base de donnée (fast_food par exemple). Puis, cliquer sur le bouton Créer

Créer une table

Taper le nom de la table dans le champ Nom (plats par exemple).

Taper le nombre de champs (3 par exemple).

Cliquer sur le bouton Créer.

Indiquer la structure de la table

Taper les noms des champs et leurs types.

Sélectionner la clé Primaire auto-incrémentée.

  • id : int [Clé primaire, autoincrémenté]
  • nom_plat : varchar(48)
  • prix : double

Faire défiler la page en bas pour cliquer sur le bouton Sauvegarder.

La table est créée. phpMyAdmin affiche le code SQL qui a permis de créer la table.

Insérer des données dans la table

Cliquer sur le lien Insérer

Remplir les champs avec les valeurs adéquates. Laisser le champ id (auto-incrémenté) vide.

Cliquer sur le bouton Exécuter.

Les enregistrements ont été bien insérés. phpMyAdmin affiche le code SQL qui a été utilisé.

Afficher les données d'une table

Cliquer sur le nom de la table pour afficher son contenu.

Cliquer sur l'icône à gauche du nom de la table pour afficher sa structure.
